# Craft-UI ‚Äî AI Component Generator

Craft-UI is a modular, production-oriented AI tool that generates responsive, modern, and animated frontend UI components from natural-language descriptions. Powered by Google GenAI, it helps developers and designers quickly prototype UI elements in multiple frameworks.


---

## ‚ú® Features

- AI-powered UI component generation from text prompts
- Monaco-based code editor with syntax highlighting
- Live preview rendering with iframe isolation
- Fullscreen preview mode
- Copy & export generated code
- Global dark / light mode with professional theming
- Responsive, modern layout inspired by production SaaS tools
- Clean state management with preserved UX flows

---

## üß± Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Code Editor:** Monaco Editor
- **Icons:** React Icons
- **State Management:** React Hooks
- **Build Tool:** Vite

---

## üéØ How It Works

1. Select a framework from the dropdown.  
2. Describe your component in detail (e.g., buttons, cards, forms).  
3. Click **Generate** to get the code.  
4. Use the code editor to **copy, download, or preview** your component live.
